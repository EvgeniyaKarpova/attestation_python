# Планер Wildberries

Приложение для учета доходов и расходов, связанных с Wildberries.

## Описание

Приложение позволяет добавлять информацию о покупках и пополнениях, просматривать историю операций и анализировать расходы по категориям.

## Функциональность

*   **Добавление операций:** Ввод информации о сумме, категории, дате, комментарии и типе операции (покупка или пополнение).
*   **Просмотр истории операций:** Таблица с историей операций, включающая тип, сумму, категорию, дату и комментарий.
*   **Анализ расходов по категориям:** Графическое представление расходов по категориям.
*   **График покупок во времени:** Отображение динамики расходов с течением времени.
*   **Топ-5 дорогих покупок:**Диаграмма, отображающая 5 самых крупных расходов.
*   **Расчет баланса:** вычисляется на основе транзакциями.

## Использование

1.  Запустите файл `qui.py`.
2.  В верхней части окна введите данные об операции.
3.  Нажмите кнопку "Добавить операцию".
4.  История операций отобразится в таблице в нижней части окна.
